OPL & Emulation Guide

This emulator acts as a faithful 9-pin impact printer using the industry standard ESC/P control set. Any system that outputs ASCII characters can print text directly.
For graphics and custom characters, you can redefine fonts dynamically using ESC & or draw bit-images using ESC K (Draft/60 DPI), ESC L (120 DPI), or ESC Z (240 DPI).

Help Reference

Getting Started

Connect your hardware or link the emulator using the CONSOLE tab. If using Web Serial, ensure baud rates match.

Use the CONSOLE text box to quickly test characters, or type escape sequences like \eE (Bold On) or \eF (Bold Off).

ESC/P Control Codes

The EFX-80 supports standard ESC/P commands:

  • ESC @ : Reset
  • ESC ! n : Master Select
  • ESC E / F : Bold On/Off
  • ESC 4 / 5 : Italic On/Off
  • ESC G / H : Double Strike On/Off
  • ESC - n : Underline On/Off
  • SI / DC2 : Condensed On/Off
  • ESC P / M : Pica / Elite Pitch
  • ESC 2 / 3 n : Set Line Spacing (1/6" / n/216")
  • ESC C n / 0 n : Page Length (Lines / Inches)
  • ESC N n / O : Skip-Over Perforations On/Off
  • ESC & : Download font character
  • ESC % n : User-defined Font On/Off
  • ESC : 0 0 0 : Copy ROM characters to RAM
  • ESC K/L/Y/Z : 8-Pin Bit Graphics (60/120/120-HS/240 DPI)
  • LF / CR / FF : Line Feed / Carriage Return / Form Feed
  • HT / CAN : Horizontal Tab / Cancel Line

Keyboard Shortcuts

Global emulator control hotkeys (when text fields are not focused):

  • F : Feed paper 3 lines
  • T : Tear off paper
  • M : Strip tractor margins
  • D : Run self-test diagnostics page
  • Delete : Clear paper roll & reset buffer
  • S : Save Printout (PNG)
  • C : Copy Printout (Image to Clipboard)
  • ↑ / ↓ : Scroll paper roll (24px steps)
  • W (Hold) + ↑ / ↓ : Micro-scrolling (1px step)

PC Connections

Classic serial devices may need custom wiring to connect the P.C. serial port connector.

If using custom connections you will need to connect at least the following PC COM port pins to the classic serial device (e.g. another classic P.C.):

  • 25 Way: pins 2, 3 & 7
  • or
  • 9 Way: pins 2, 3 & 5

Depending on the arrangement of the device being connected you may need 'a cross over' connection. e.g. pin 2 of one device is connected to pin 3 of the other device.

Note: A classic Organiser II COMMS/RS232 link is designed for direct connection to a 25 way D-Type COM port connection.

Hardware & Conflict Warnings

A serial COM port i.e. a Serial-to-USB convertor or internal serial COM port hardware can only be connected to one application on the host machine.

Connect your legacy serial device e.g. the Organiser's serial COMMS Link to the selected serial COM port or Serial-to-USB convertor on the host computer.

The browsers Web Serial will try to set the default baud rate settings. You may need to set your host computers serial port settings to match ( e.g. 9600 Baud, 8 Data Bits, No Parity, and 1 Stop Bit ).

Important: Ensure no other software (terminal programs, ORG-LINK or OPK transfers) is holding the serial port open, as standard P.C. port hardware can only be accessed by one app / program at a time.
